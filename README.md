Project Overview

This repository contains the end-to-end data analysis workflow for Bright TV, a premier
broadcasting platform in South Africa. The project focuses on transforming raw viewership
data into actionable business intelligence to drive advertising strategy and content
scheduling.

Key Achievement: Identified a "Youth-Dominant" demographic profile (78% Early

Adulthood) and localized a "Payday Peak" (34.5% engagement increase) at month-
end.

Data Architecture

The pipeline utilizes a structured approach to process television viewership trends and age
demographics:
▹ Data Source: bright_tv_dataset_viewership
▹ Storage:  Databricks warehousing for scalable SQL query optimization.
▹ Visualization: Strategic mapping of data flows using Miro and deliverable planning via
Gantt charts. presentation planing vai canva.

Core Insights

1. Demographic Identity
The audience is primarily Early Adulthood (77.6%), concentrated heavily in Gauteng.
This demographic profile aligns perfectly with the high engagement observed in
SuperSport Live and Channel O.
2. Temporal Patterns
Viewership follows a "Crescendo Pattern," building throughout the month to a 34.47% peak
at **Month-End**. Weekly peaks are concentrated on Fridays and Saturdays.
3. Technical Audit (Data Hygiene)
Identified a consistent 9-10% Null Rate across daily, channel, and age metrics. Addressing
this tracking gap represents a significant opportunity for improved audience attribution.
